# Contributing to Claudo

Thank you for your interest in contributing to Claudo.

Claudo is currently focused on architecture, design, and security principles
for AI-assisted execution systems.

## How to Contribute

- Open an issue to discuss proposed changes
- Keep contributions aligned with the project's design goals
- Avoid implementation claims that are not yet supported

## Scope

This repository does not currently include production executors,
autonomous agents, or fund-handling logic.

Such components, if any, will exist in separate repositories.
